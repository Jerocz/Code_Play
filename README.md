# CodeTutor 🚀

Proyecto educativo gamificado para aprender programación.

## Backend
Tecnologías:
- FastAPI
- Uvicorn

## Frontend
- HTML
- CSS
- JavaScript

---

## Instalar dependencias

```bash
pip install -r requirements.txt
```

---

## Ejecutar backend

Entrar a la carpeta backend:

```bash
cd backend
uvicorn main:app --reload
```

---

## Abrir frontend

Abrir:
frontend/index.html
